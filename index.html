Optimum Stream Remote Setup Guide

Remote Pairing Assistant

A simple step-by-step guide to pair your Optimum Stream remote with your TV.

Step 1

Prepare Your Remote

  1. Turn on your TV
  2. Remove the back cover of your remote
  3. Insert two AAA batteries following the (+/-) markings
  4. Replace the back cover until it clicks
Step 2

Power Up

  1. Position your Stream box where the remote can point directly at it
  2. Press the Power button (top-right) on your remote
  3. Wait for the Stream box to Boot up
  4. Press the Home button (below navigation pad)
Step 3

Access Settings

  1. Find the Settings icon (gear/cog symbol) on screen
  2. Use the navigation pad to highlight Settings
  3. Press the Select button (center of pad)
  4. Navigate to "Device & Remote Settings"
Step 4

Pair Remote

  1. Select "Remotes & Accessories"
  2. Choose "Add Stream Remote"
  3. Point remote at the Stream box
  4. Press and hold Select + Mute buttons
  5. Release when the remote light blinks
Step 5

Configure TV Control

  1. Return to "Device & Remote Settings"
  2. Select "Stream Remote"
  3. Choose "Change TV Setup"
  4. Test Volume Up/Down buttons
  5. Select "Yes" if volume changes
  6. If no response, select "No" and pick your TV brand

Thank You!

Your remote is now set up and ready to use.
